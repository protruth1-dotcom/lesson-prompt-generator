# Islamic Studies Specification — Safar Academy Aligned

## Core Requirements

All Islamic Studies lessons generated by the prompt must follow these rules:

### Language & Script
- **Arabic script** is used for all main topics, subtopics, and key Islamic terms — no transliteration anywhere
- **No transliteration** — never include phonetic Latin-script representations of Arabic words
- English translation is provided alongside Arabic where needed for comprehension
- The prompt must explicitly instruct the AI: "Use Arabic script for all Islamic terms, topic names, and keywords. Do NOT include transliteration."

### Quran References
Every time a Quran verse is referenced, the prompt instructs the AI to present it in this exact format:

```
Arabic Ayah: [Full Arabic text of the verse]
Translation: [English translation — Mustafa Khattab, The Clear Quran]
Reference: [Surah Name, Ayah Number]
```

The prompt must include: "For all Quran translations, use ONLY 'The Clear Quran' by Dr. Mustafa Khattab as the translation source. Cite the surah name and ayah number."

### Dua References
Every dua must be presented as:

```
Arabic: [Full Arabic text of the dua]
Translation: [English translation]
Source: [Hadith collection — e.g., Sahih al-Bukhari, Sahih Muslim, Sunan Abu Dawud, Jami' at-Tirmidhi, Sunan an-Nasa'i, Sunan Ibn Majah, Riyad as-Salihin]
```

The prompt must include: "For all duas, provide the Arabic text and English translation from authentic Sunni hadith collections. Always cite which hadith collection the dua is sourced from."

### Character Building Takeaway
Every Islamic Studies lesson ends with: **"Something I can practice this week:"** — a specific, actionable, age-appropriate behavior drawn from the lesson content.

### Vocabulary for Islamic Studies
The Key Vocabulary Box in Islamic Studies lessons must include:
- The Arabic term (in Arabic script)
- The English meaning
- A one-sentence example using the concept

Example:
```
التَّوَكُّل — Reliance on Allah — When you study hard for a test and then trust that Allah will help you, that is التَّوَكُّل.
```

## Islamic Studies Block (Injected into Prompt Header)

When Islamic Studies is selected, the following block is added to the top of the generated prompt, after the role instruction:

```
IMPORTANT ISLAMIC STUDIES REQUIREMENTS — Follow these exactly:

1. Use Arabic script (not transliteration) for ALL Islamic terms, topic names, subtopics, and keywords throughout the lesson and quiz.
2. For ALL Quran verses, use ONLY "The Clear Quran" by Dr. Mustafa Khattab as the translation source. Present each verse as:
   - Arabic ayah (full text)
   - English translation (Mustafa Khattab)
   - Surah name and ayah number
3. For ALL duas, provide:
   - Arabic text (full)
   - English translation from an authentic Sunni hadith source
   - Name of the hadith collection it is sourced from (e.g., Sahih al-Bukhari, Sahih Muslim, Jami' at-Tirmidhi, Sunan Abu Dawud)
4. All Islamic teachings must be from a Sunni (أهل السنة والجماعة) perspective.
5. End the lesson with a "Something I can practice this week:" takeaway — one specific, actionable behavior the student can apply.
6. Connect teachings to real daily-life situations a 4th/5th grader would encounter.
7. Never include transliteration (Latin-script phonetic spelling of Arabic). Arabic terms appear in Arabic script only, with English translation provided where needed.
```

## Visual Strategy for Islamic Studies

Most Islamic Studies topics are **text-heavy** and use **Tier 1 visuals** only:

| Topic Type | Visual Approach |
|------------|----------------|
| Fiqh (Wudu, Salah) | Flowchart showing steps in order |
| Seerah (Timeline) | Timeline diagram of events |
| Aqeedah (Categories) | Classification chart / concept map |
| Akhlaq (Scenarios) | No visual needed — scenario-based text |
| Quran (Ayat) | Arabic text displayed prominently — no image needed |
| Duas | Arabic text displayed prominently — no image needed |
| Islamic History (Prophets) | Timeline of events in the prophet's story |

Tier 2 visuals (generated images) are **rarely needed** for Islamic Studies. If a topic does require an image (e.g., a map of the Hijrah route), the prompt uses the standard Tier 2 block with search sources.

## Quiz Adjustments for Islamic Studies

- **Vocabulary questions** use Arabic terms with English definitions (matching format)
- **Scenario-based questions** are especially important — present real-life situations a student might face and ask them to apply the Islamic value or ruling
- **Quran-based questions** may ask the student to identify which surah a lesson is from, or what the main message of an ayah is (NOT memorization of Arabic text in the quiz — comprehension only)
- **Dua questions** may ask what dua is appropriate for a given situation (e.g., "What dua should you say before eating?")
- **True/False with explain** works well for Aqeedah and Fiqh (e.g., "Tayammum can be used when water is available. True or False? Explain.")
